# Tiny Tails Rescue

`index.html` をブラウザで開くと、**フィールド探索型ペットレスキューアドベンチャー**を遊べます。Three.js による 3D で、明るい Roblox 風のビジュアルです。

## 遊び方

1. **Hub（拠点）** からスタート
2. 緑のゲートで **草原エリア** へ出発
3. フィールドの **茂み・ケージ・岩** に近づいて **Space 長押し** で動物を救助
4. 敵（ハチ・カラスなど）に触れると **ノックバック**（救助もキャンセル）
5. 光るポータルに戻って **[E]** で拠点へ帰還 → コイン獲得
6. **ショップ** で速度UP・ツール強化・容量拡張
7. 10匹救助 or 200コインで **湖畔エリア** を解放

## 操作

| 操作 | キー |
|------|------|
| 前進 / 後退 | W/S / ↑↓ |
| 旋回（左/右） | A/D / ←→ |
| インタラクト | E |
| 救助（長押し） | Space |
| カメラ回転（キャラも旋回） | マウス右ドラッグ |
| ズーム | マウスホイール |

モバイルでは左側ジョイスティック（上下=前後、左右=旋回） + 右下レスキューボタンで操作できます。

## ゲーム内容

- **Hub**: 保護施設（ペット図鑑）、ショップ、2つのワールドゲート
- **草原**: 子犬・うさぎ・子ぶた・子猫（救助ノード10個、敵3体）
- **湖畔**: アヒルの子・ビーバー・カワウソ・カメ（救助ノード10個、敵3体）
- **レア度**: コモン / アンコモン / レア / エピック（光・サイズ変化）
- **ペット装備**: 1匹を連れ歩き（移動速度+10%）

## データ保存

`localStorage`（キー `tiny_tails_rescue_v1`）に自動保存されます。

## 企画の出自

仕様書「Roblox風ペットレスキューゲーム 調査・再構成仕様書」に基づく Ver2 実装です。
