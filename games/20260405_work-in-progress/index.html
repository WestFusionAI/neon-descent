ペットレスキュー

災害が下から迫るビルで、ペットを助けてヘリで脱出するデモ版です。上の階ほどレアな子がいますが、危険も早まります。

遊び方
  1. ミッションを選ぶ → ステージ（地震など）をタップ。
  2. 水色の丸が隊員。右端の縦線が階段。右端まで行って ↑（または画面の ↑ ボタン） で上の階へ。
  3. ペットの絵文字に近づくと自動で救出。
  4. 下から災害が迫る前に ヘリで脱出 を押すとクリア（デモ）。

起動：この index.html をブラウザ（Chrome / Edge など）で開いてください。フォルダをダブルクリックするだけでは開けない場合は、ファイルをブラウザのウィンドウにドラッグ＆ドロップしてください。

出動ステージ

5種類の災害。デモでは挙動は共通で、見た目と速度倍率だけ変えています。

ショップ（デモ）

コインでアイテムを買うと、ミッション中に「投げる」で使えます（デモは1種類のみ接続）。所持: 0 コイン
