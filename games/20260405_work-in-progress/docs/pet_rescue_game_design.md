# ペットレスキュー - ゲーム企画書

## 1. ゲーム概要

### コンセプト
プレイヤーは「ペットレスキュー隊」の一員として、様々な災害現場に出動し、取り残されたペットたちを救出するゲーム。

### 基本ルール
- **ゲーム開始**: 建物（ホテル、家、マンション）の外からスタート
- 建物の中に入っていき、ペットを探す
- 下から災害（崩壊、津波、火、怪獣など）が迫ってくる
- 上の階へ逃げながらペットを救出する
- 危険になる前にヘリコプターで脱出すれば成功
- ビルが壊れてしまったら救出失敗
- 欲張って上に行くほどレアなペットが手に入るが、危険度も上がる

### 建物の階数（リアルな設定）
- **小さな家**: 3階建て
- **マンション**: 10階建て
- **大きなホテル**: 20階建て
- ステージによって建物の規模が異なり、高い建物ほど上層に超レアなペットがいる

## 2. ゲームの流れ

```
依頼を受ける
    ↓
建物の外からスタート（ホテル/マンション/家の外観が見える）
    ↓
建物の中に入っていく
    ↓
災害発生！（5種類のステージから）
    ↓
ペットを救出しながら上の階へ（制限時間あり）
    ↓
脱出タイミングを判断
 - 危険になる前に脱出？
 - もっと上の階を目指す？
    ↓
┌─────────────────┬─────────────────┐
│   救出失敗      │    救出成功     │
│  ビル崩壊...    │  ヘリで脱出！   │
│                 │  報酬ゲット     │
└─────────────────┴─────────────────┘
    ↓
次のミッションへ（繰り返し）
```

## 3. ステージ一覧（5種類の災害）

### ステージ1: 地震
- **場所**: 3階建ての大きな家
- **開始**: 家の外からスタート → 中に入る
- **特徴**: 外側から崩れてくる
- **攻略**: 崩れてくるところを逃げながら、どんどん奥（上の階）へ進む
- **難易度上昇**: 上の階ほどレアなペットがいる

### ステージ2: 怪獣
- **場所**: 10階建てのビルディング
- **開始**: ビルの外からスタート → 中に入る
- **特徴**: 怪獣が建物を襲ってくる
- **攻略**: 怪獣に襲われる前にペットを救出して脱出
- **難易度上昇**: 上の階に行くほど怪獣が早くなる

### ステージ3: 津波
- **場所**: 20階建ての大きなホテル
- **開始**: ホテルの外からスタート → 中に入る
- **特徴**: 水が下から上がってくる
- **攻略**: 水中をタップで泳ぎ、ベッドなどの障害物を左右で避けながら上へ移動
- **難易度上昇**: 水位上昇スピードが速くなる
- **備考**: 20階まであるので、最上階には神話級のレアペットがいる

### ステージ4: 火事
- **場所**: 10階建てのマンション
- **開始**: マンションの外からスタート → 中に入る
- **特徴**: 炎が迫ってくる
- **攻略**: 炎から逃げながらペットを救出

### ステージ5: クレーン（悪者）
- **場所**: 15階建ての建設現場またはビル
- **開始**: 建物の外からスタート → 中に入る
- **特徴**: 悪者がクレーンを操作して追いかけてくる
- **攻略**: クレーンから逃げながらペットを救出
- **備考**: 自然災害ではなく人為的な脅威

## 4. ペットのレア度システム

建物の階層によって出現するペットのレア度が変わる。
建物が高いほど（ホテル20階など）、より上層でさらにレアなペットが出現。

### 階層別ペット一覧

| 階層 | レア度 | 出現ペット |
|------|--------|------------|
| 1〜2階 | コモン | ハムスター、ハリネズミ（小動物） |
| 3〜5階 | ノーマル | 犬、猫、鳥、うさぎ（普通のペット） |
| 6〜8階 | レア | 光る犬、光る猫（キラキラ進化版） |
| 9〜12階 | 超レア | 光るペット＋羽が生えている |
| 13〜16階 | 伝説級 | ドラゴン、光る狼、巨大レッサーパンダ |
| 17〜20階 | 神話級 | 究極進化ペット（下記参照） |

### ペットの進化パターン（詳細）

#### ノーマル → レア
- 通常の犬・猫 → 体が光っている

#### レア → 超レア
- 光っている → 光っている＋羽が生えている

#### 超レア → 伝説級
- 狼、ドラゴンなど特別な動物が登場
- めっちゃ光っている
- 羽がついている

#### 伝説級 → 神話級（最高レア）
- **羽つき犬**: 羽が生えた犬
- **親子ペット**: 赤ちゃんも一緒についてくる犬・猫
- **巨大進化**: めっちゃ大きく進化したレッサーパンダ
- **フル装備**: 光っている＋羽＋赤ちゃん付き
- **虹色ドラゴン**: 全身が虹色に輝くドラゴン

### レアペットの特徴組み合わせ

| 特徴 | 説明 |
|------|------|
| 光っている | 体がキラキラ光る |
| 羽がついている | 背中に羽が生えている |
| 赤ちゃん付き | 小さい赤ちゃんペットが一緒についてくる |
| 巨大化 | 通常より大きいサイズ |
| 虹色 | 全身が虹色に輝く |
| 王冠付き | 頭に王冠をかぶっている |

**組み合わせ例**:
- 光る羽つき犬
- 赤ちゃん連れの光る猫
- 巨大な虹色ドラゴン＋赤ちゃんドラゴン付き

## 5. 操作方法

### 基本操作（Googleマップ風）

| 操作 | アクション |
|------|------------|
| スワイプ | キャラクターの向きを変える |
| タップ | 向いている方向に前進する |

### 津波ステージ専用操作
| 操作 | アクション |
|------|------------|
| タップ | 水中を泳ぐ |
| 左右スワイプ | 障害物（ベッドなど）を避ける |

### アイテム使用
| 操作 | アクション |
|------|------------|
| 「投げる」ボタンをタップ | 手に持っているアイテムを投げる |

- アイテムは購入後、手に持っている状態になる
- 「投げる」ボタンでアイテムを発射
- アイテムがどこかに当たると効果が発動する

## 6. ショップシステム

### 購入可能アイテム

| アイテム名 | 効果 |
|------------|------|
| 遅くするマシン | 怪獣や災害のスピードを遅くする |
| 時間延長 | 救出に使える制限時間を増やす |
| 大きなケージ | 一度に運べるペットの数がアップ |
| 新しい車 | レスキュー用の車両（詳細は要検討） |

### アイテムの使い方
1. ショップでアイテムを購入
2. ミッション中、アイテムを手に持った状態
3. 使いたいタイミングで「投げる」ボタンをタップ
4. アイテムが飛んでいき、当たった場所で効果発動

### プレイヤーキャラクター
- 人間のペットレスキュー隊員

### 服の進化

| 段階 | 服の種類 | 入手方法 |
|------|----------|----------|

| 3段階目 | 超かっこいい服 | さらに高額で購入 |
| ... | さらに進化 | どんどんアップグレード可能 |

## 8. 報酬システム

### 報酬の獲得方法
1. **ペット救出ポイント**: 助けたペットの数に応じてポイント獲得
2. **飼い主からのお礼**: 救出したペットの飼い主から「ありがとう」とお金をもらえる

### 報酬の使い道
- ショップでアイテム購入
- 新しい服の購入
- 装備のアップグレード

---

## 9. 難易度バランス

### 上の階に行くメリット
- よりレアなペットを救出できる
- より多くのポイント・報酬を獲得できる

### 上の階に行くリスク
- 災害（マグマ、津波、怪獣など）がより早くなる
- 危険度が増す
- 脱出に失敗する可能性が高まる

### プレイヤーの判断
- 「もう少し上に行ってレアなペットを狙う」か
- 「ここで安全に脱出する」か
- このリスク・リワードの判断がゲームの醍醐味

---

## 10. 脱出方法

### 基本ルール
- 各フロアからヘリコプターで脱出可能
- 最上階まで行かなくても脱出できる
- ただし、上に行けば行くほど：
  - よりレアなペットを救出できる
  - より多くのペットを救出できる

### 脱出成功条件
- 災害に巻き込まれる前にヘリコプターに乗る

### 脱出失敗条件
- 津波に飲まれる
- 怪獣に襲われてビルが壊れる
- 崩壊に巻き込まれる
- 火に囲まれる
- クレーンに捕まる

## 12. まとめ

**ペットレスキュー**は、災害からペットを救出するシンプルながら奥深いゲーム。

**コアループ**:
救出 → 報酬獲得 → ショップで強化 → より難しいステージへ挑戦

**魅力ポイント**:
- リスク vs リワードの判断が楽しい
- ペットの進化・レア度システム
- キャラクターのカスタマイズ（服装）
- 5種類の異なる災害ステージ
- 直感的な操作方法
