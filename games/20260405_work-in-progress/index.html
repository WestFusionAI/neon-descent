ディザスター・ペット・レスキュー

Disaster Pet Rescue · デモ版

救出アクション × 育成シミュレーションのたたき台です。災害からペットを連れ帰り、拠点で餌やけをして変異・進化のきっかけを試せます。

遊び方（デモ）
  1. レスキュー：ステージへ出動。初めての方は一覧先頭の 操作デモ（練習） がおすすめです。本番は下から災害が迫り、リスク％も時間で上昇。100%で建物崩壊。
  2. 一部ステージは 鍵 を拾わないと上の階へ進めません（3Dで金色の鍵）。
  3. メインは 3Dビュー。隊員は ←→ で左右、W/S または 手前・奥 で奥行き移動。上の階は右奥の階段ゾーンで ↑ のみ（横移動だけでは上がれません）。
  4. 視点：PC では Q / E で回転、右ボタンドラッグで回転・俯仰、マウスホイールでズーム。タッチは画面上部 HUD の 視点◀ / ▶視点 で回転。
  5. 右下の 断面モニタ で、縦割りの建物に対する隊員位置（青）と災害の高さが分かります。
  6. 脱出で成功。救出した子は 拠点（水槽・飼育スペース） に送られ、餌で変異が進むことがあります。
  7. ショップでコイル・餌・拠点拡張などを購入。

起動：index.html を Chrome / Edge 等で開いてください。

災害ステージ

火災・水没・氷山などテーマ別。プレイは Three.js 3D。速度・リスク・鍵の要否が異なります。

レスキュー拠点

救出したペットを飼育。餌で突然変異（銀・ダイヤ・エメラルド…）が進むことがあります。所持コイン: 0
