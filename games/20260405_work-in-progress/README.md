# ディザスター・ペット・レスキュー（Disaster Pet Rescue）— デモ

`index.html` をブラウザで開くと、**救出アクション × 育成シミュレーション**のたたき台を遊べます。メインのプレイ画面は **Three.js による 3D**（**奥行き移動・階段ゾーン・オービット視点**あり）。隊員は各階の**床面に合わせた高さ**に立ち、HUD の **視点·斜め／横／上**（またはキー **1／2／3**）で**横から・見下ろし**などに切り替え可能です。右下の **断面モニタ（2D）** も併用してください。ビジュアルは **Roblox 的**な明るいプラスチック調を意識しています。リポジトリ直下の [ゲーム一覧](../../index.html) からもリンクされています。

## 企画書（要件）

- 詳細なルール・5 災害・ペットレア度などは [docs/pet_rescue_game_design.md](./docs/pet_rescue_game_design.md) を参照（末尾に **デモ実装との対応メモ** あり）。
- デモでは建物は **ステージごとの `floorCount`（3〜8 階）** に圧縮し、カードに **企画の階数** と併記しています。

## コンセプト（要件の整理）

- 迫りくる災害の**制限時間とリスク**のなかでペットを救出し、**拠点（水槽・飼育スペース）**で育てて**変異・進化**のきっかけを試すデモです。
- **コアループ（デモで触れる範囲）**
  - **レスキュー**: 災害の上昇・**リスク％**（100%で崩壊）・一部ステージの**鍵（🗝）**探索
  - **拠点**: 救出成功したペットをリスト表示し、**餌（リンゴ・プランクトン・骨・肉）**でランダムに変異段階が進むことがある
  - **ショップ**: コインで**スピードコイル**・**遅くするマシン**・**拠点拡張**・**餌**を購入

ステージは企画書の **5 災害**に対応したテーマ（**地震・怪獣・津波・火事・クレーン**）＋先頭の **操作デモ（練習）**。3D フィールドの見た目は共通で、**速度・リスク・鍵・階数**がステージごとに変わります。

## 企画の出自（記録として）

- **ゲームのアイデア・世界観の中心は、制作者の娘の発想です。** 親としてこのリポジトリに載せているだけであり、**親個人のオリジナル成果として名乗るものではありません。**
- 親子で会話しながらアイデアを膨らませ、**音声メモに残した内容を文字起こし**し、それを **AI に要件整理してもらったうえで**、ここにある形にゲーム化しました。
- **「子どもと話したことを、音声→テキスト→要件→プロトタイプまでつなげられる」** というプロセスそのものも、試みの記録として残しています。

## データの保存

進行はブラウザの `localStorage`（キー `disaster_pet_rescue_demo_v2`）に保存されます。旧キー `pet_rescue_demo_v1` からコイン等を引き継ぐ簡易移行があります。
